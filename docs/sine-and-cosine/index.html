Source snapshot

The local route vendors the published Sine and Cosine page HTML, stylesheet, and script, then patches asset paths, removes public-shell chrome, and replaces the deprecated remote MathJax load with a local vendor copy.

  • Live original verified against https://setosa.io/ev/sine-and-cosine/ on 2026-03-14.
  • The local copy removes EV branding, bylines, share widgets, newsletter prompts, analytics, and comments.
  • The published page still points at a retired MathJax CDN URL, so the replica vendors MathJax 2 locally under ev/scripts/mathjax/.

Shared EV layer

  • ../ev/scripts/d3.js, angular.js, and common.js are reused unchanged.
  • ../ev/styles/style.css remains the common EV base stylesheet.
  • sine-and-cosine/style.css adds narrow-screen wrappers and display handling for local MathJax output.

Asset and source map

  • sine-and-cosine/index.html owns local metadata, local MathJax wiring, footer wiring, and the cleaned public shell.
  • sine-and-cosine/script.js keeps the published controllers and directives for similar triangles, trig transforms, and linked coordinates.
  • sine-and-cosine/style.css keeps the original visual language and adds responsive wrappers.
  • ev/scripts/mathjax/ contains the vendored MathJax runtime needed for the Taylor-series block.

Known deviations

  • The public route no longer shows EV branding, creator credits, share widgets, newsletter forms, analytics, or comments.
  • The live page verified on 2026-03-14 does not ship a single page-wide angle controller or pause/reset transport. Instead it has a draggable similar-triangle view, separate sine and cosine autoplay panels, and a linked-coordinate sandbox; the replica preserves that exact surface.
  • The local route swaps the retired remote MathJax load for a local vendor copy but keeps the same inline tex2jax configuration and displayed Taylor-series content.
