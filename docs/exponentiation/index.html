Source snapshot

The local route vendors the published Exponentiation page HTML, stylesheet, and inline script, then patches asset paths and public-shell chrome so the original EV runtime works under the /interactive-explanation/ subpath.

  • Live original verified against https://setosa.io/ev/exponentiation/ on 2026-03-14.
  • The local copy removes EV branding, bylines, share widgets, newsletter prompts, analytics, and comments.
  • The controller logic and directives remain inline, matching the published page structure instead of being extracted for v1.

Shared EV layer

  • ../ev/scripts/d3.js, angular.js, and common.js are reused unchanged.
  • ../ev/styles/style.css stays the common EV base layer.
  • exponentiation/style.css now adds lightweight overflow handling for the wide growth and virus demos.

Asset and source map

  • exponentiation/index.html owns local metadata, footer wiring, and the published inline Angular runtime.
  • exponentiation/style.css keeps the original custom-element sizing and adds local scroll wrappers.
  • exponentiation/fb-thumb.png is vendored for subpath-safe metadata and sharing previews.

Known deviations

  • The public route no longer shows EV branding, author credits, share widgets, newsletter forms, analytics, or comments.
  • The published live page verified on 2026-03-14 does not ship a standalone comparison panel, so the replica keeps the same simple, linear, exponential, and virus sections instead of inventing one.
  • The live HTML also ships the linear demo rate control as a broken <selct> typo; the local route repairs that one tag so the intended rate dropdown works.
