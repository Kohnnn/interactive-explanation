Source snapshot

The local route vendors the published page HTML, stylesheet, script, map data, and required shared scripts, then patches pathing and public-shell markup so the original Angular runtime works under the /interactive-explanation/ subpath.

  • Live original verified against https://setosa.io/ev/eigenvectors-and-eigenvalues/ on 2026-03-14.
  • The local copy removes EV branding, creator bylines, share widgets, newsletter prompts, analytics, and comments.
  • The main controller and directive logic remain in the published eigenvectors-and-eigenvalues/script.js file for v1.

Shared EV layer

  • ../ev/scripts/d3.js, angular.js, and common.js are reused as the main runtime layer.
  • ../ev/scripts/topojson.js is vendored locally for the migration-map scene.
  • ../ev/scripts/mathjax/MathJax.js replaces the retired CDN dependency used by the published page.

Asset and source map

  • eigenvectors-and-eigenvalues/index.html owns the cleaned public shell, metadata, local Markov cross-link, and footer wiring.
  • eigenvectors-and-eigenvalues/script.js keeps the published controllers and directives for the basis, Fibonacci, migration, and spiral scenes.
  • eigenvectors-and-eigenvalues/resources/us.json supplies the local map geometry for the steady-state migration demo.
  • eigenvectors-and-eigenvalues/style.css preserves the original styling and adds only narrow-screen overflow helpers.

Known deviations

  • The public route no longer shows EV branding, creator links, share widgets, newsletter forms, analytics, or comments.
  • The published Markov cross-link is rewritten to the local ../markov-chains/ route so the narrative stays inside the replica site.
  • MathJax now loads from the local EV asset tree, and the US map JSON loads from ./resources/us.json instead of the original site root.
