Source snapshot

The local route vendors the published OLS page shell, route stylesheet, page bundle, and shared EV runtime, then patches local asset paths and public page chrome so the runtime works cleanly under /interactive-explanation/.

  • Live original verified against https://setosa.io/ev/ordinary-least-squares-regression/ on 2026-03-18.
  • The local copy removes EV branding, bylines, share widgets, newsletter prompts, analytics, and comments.
  • The interactive runtime still relies on the published React, D3, Three.js, and shared EV bundle shipped by the original page.

Shared EV layer

  • ../ev/styles/style.css remains the shared Setosa EV visual baseline for typography, layout, and menu treatment.
  • ../ev/_build/js/common-shared.js is vendored locally in the same relative shape the published page expects, avoiding any live-root dependency.
  • ordinary-least-squares-regression/_bundle.js was patched only for local asset URLs and footer-policy cleanup around the published runtime.

Asset and source map

  • ordinary-least-squares-regression/index.html owns the local metadata shell, footer hook, docs link, and relative bundle paths.
  • ordinary-least-squares-regression/style.css preserves the route-specific nob and heading styling shipped by the original page.
  • ordinary-least-squares-regression/_bundle.js keeps the original scatterplot, squared-error, parameter, and 3D loss-surface interactions.
  • ordinary-least-squares-regression/resources/*.gif and fb-thumb.png were vendored locally for subpath-safe static hosting.

Known deviations

  • The public route no longer shows EV branding, author credits, share widgets, newsletter forms, analytics, or comments.
  • The route adds a small docs link and shared footer, but leaves the published runtime intact.
  • The original “special thanks” paragraph was localized into neutral public copy so provenance stays in docs only.
