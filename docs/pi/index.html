Source snapshot

The local route vendors the published Pi page HTML, stylesheet, and script, then patches asset paths and the surrounding public shell so the original Angular runtime works under the /interactive-explanation/ subpath.

  • Live original verified against https://setosa.io/ev/pi/ on 2026-03-14.
  • The local copy removes EV branding, bylines, share widgets, newsletter prompts, analytics, and comments.
  • The main interactive logic remains in the original pi/script.js file for v1.

Shared EV layer

  • ../ev/scripts/d3.js, angular.js, and common.js are reused unchanged.
  • ../ev/styles/style.css remains the common EV base stylesheet.
  • pi/style.css adds only local overflow and control-layout rules for responsive hosting.

Asset and source map

  • pi/index.html owns local metadata, footer wiring, and the cleaned public shell.
  • pi/script.js keeps the published evt, circle-demo, and pi-demo directives.
  • pi/style.css keeps the original demo styling and adds narrow-screen wrappers.
  • pi/fb-thumb.png is vendored for local metadata and preview safety.

Known deviations

  • The public route no longer shows EV branding, author credits, share widgets, newsletter forms, analytics, or comments.
  • The live Pi page verified on 2026-03-14 only ships the circle demo and the wrap/size pi demo, so the replica preserves that exact surface instead of inventing extra area-decomposition scenes.
  • The public route wraps the custom elements for horizontal scrolling on narrow screens, but does not refactor the original geometry logic.
