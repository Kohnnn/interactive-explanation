Source snapshot

The local route vendors the published article HTML, shared article CSS, page-local CSS, scripts, WebGL textures, and article images, then patches only the subpath-hosting shell and provenance surfaces needed for the public replica.

  • Live original verified against https://ciechanow.ski/color-spaces/ on 2026-03-20.
  • The local copy removes the upstream banner, blog navigation, analytics, Patreon prompt, and social links from the public body.
  • The published slider, matrix, 3D plot, and gamut interactions remain driven by the vendored scripts rather than a local rewrite.

Asset handoff

  • color-spaces/index.html owns the local metadata shell, shared footer hook, and relative article asset paths.
  • color-spaces/css/base.css and color-spaces/css/color_spaces.css preserve the published article layout and interaction styling with subpath-safe image URLs.
  • color-spaces/js/base.js and color-spaces/js/color_spaces.js keep the original slider, 3D plot, WebGL gamut, and chromaticity scene logic.
  • color-spaces/images/ contains the article-local diagrams, helper sprites, and gamut textures required by the published page and scripts.

Known deviations

  • The public route no longer includes the author banner, Patreon prompt, X/Twitter, Instagram, e-mail, RSS, or Google Analytics surfaces.
  • The local copy uses local fallback font stacks instead of runtime Google Fonts requests so the route stays self-contained.
  • The shared footer is the only public original-page reference surface on the shipped route.

Validation guidance

  • Keep the article structure intact; future edits should patch the published shell, not rewrite the interactions into a condensed local version.
  • Extend the shared smoke bundle instead of creating a one-off validator so later Ciechanowski routes can reuse the same family path.
  • Run both node interactive-explanation/tools/check-public-surface.mjs and node interactive-explanation/tools/smoke-bundle.mjs after touching article shell copy or any vendored runtime files.
