Image Kernels

Interactive explainer

An image kernel is a small matrix used to apply effects such as blurring, sharpening, outlining, or embossing. Kernels also show up in machine learning for feature extraction and other convolution-style operations.

Start with a black-and-white image. The matrix on the left contains brightness values between 0 and 255 for each pixel in a face image. The enlarged view makes the arithmetic easier to inspect.

Let's walk through applying the following 3×3 {{selectedKernel}} kernel to the face from above.

For each 3×3 block of pixels in the image on the left, multiply each pixel by the corresponding kernel entry and sum the results. That sum becomes a new pixel in the image on the right. Hover over a pixel on either image to see how the value is computed.

One subtlety is what to do at the edges. The top-left pixel only has three neighbors inside the image. This demo handles those missing values by treating the out-of-bounds region as black, which makes the boundary behavior easy to inspect.

Here's a playground where you can switch kernel presets, edit the 3×3 matrix directly, and see the result on the example image. The published page also supports uploading your own image and, where supported, live video.

The sharpen kernel emphasizes differences in adjacent pixel values, making the image look more vivid.

The blur kernel de-emphasizes differences in adjacent pixel values.

The emboss kernel gives the illusion of depth by emphasizing pixel differences along a direction from the top left to the bottom right.

The identity kernel leaves the image unchanged.

The custom kernel is whatever you build in the editor.

Sobel kernels highlight only the differences in adjacent pixel values in a chosen direction.

The outline kernel highlights large local differences, turning edges bright while leaving flat regions dark.
